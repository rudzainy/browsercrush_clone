# Browser Crush Clone
This is a clone of http://www.browsercrush.com/

Built using Ruby, HTML, CSS and Javascript.

## Setup
1. Download the source files.
2. Run `$ ruby app.rb`.
3. Open a web browser and type in `localhost:4567` in the address bar.
4. Enjoy the game!
